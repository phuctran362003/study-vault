## SDK là gì?

- Là một bộ kit dành cho lập trình viên dùng để chạy các file `exe` và `dll`. Cùng với đó là các “bộ đồ chơi”.
- Thuật ngữ cần nhớ: **SDK** (Software Development Kit).

## .NET SDK là gì?

- Thuật ngữ cần nhớ: **.NET SDK**.

## Quy ước về nơi lưu trữ Source Code – Project

1) **Không đặt tên thư mục/folder/directory** chứa code có dấu tiếng Việt, ký tự **Unicode**, có **khoảng trắng**, **ký tự đặc biệt** – vì nó sẽ ảnh hưởng đến quá trình **biên dịch** và đóng gói app (nhiều tool không thân thiện với Unicode/tiếng Việt có dấu).
2) **Không lưu code ở ổ đĩa `C:`**. Nên lưu ở ổ `D:` hoặc `E:`.
3) **Cấu trúc thư mục gợi ý** chứa source code:

```text
NET1702/Session01-HelloWorld/...
```

## CLI commands (.NET Command Line Interface)

Thuật ngữ cần nhớ: **CLI** (Command Line Interface).

- New project using CLI:

```bash
dotnet new console --framework net6.0 --use-program-main
```

- Build:

```bash
dotnet build
```
