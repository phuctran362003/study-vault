# Verbatim

## What You Type Is What You Get

`@` + **String** tạo ra **verbatim string literal**: giữ nguyên ký tự gõ vào, không cần thêm escape cho dấu gạch chéo ngược.

- Giữ nguyên dấu xuống dòng và dấu cách.
- Muốn thêm dấu ngoặc kép bên trong, nhân đôi ký tự `"` thành `""`.
- Hữu ích khi viết đường dẫn Windows (`C:\path`) hoặc nội dung JSON/XML nhiều dòng.

```csharp
var rawPath = @"C:\\Program Files\\dotnet";
var multiLine = @"SELECT *
FROM Products
WHERE IsActive = 1";
```

> Từ C# 11 có **raw string literal** với dấu `"""`, nhưng `@` vẫn phổ biến vì dễ đọc và hỗ trợ trong mọi phiên bản .NET.

### Khi dùng `@` với định danh

- `@` có thể đứng trước tên biến để dùng **keyword** làm tên định danh: `var @class = "Sample";`.
- Thực tế ít dùng, nhưng hữu ích khi làm việc với thư viện sinh code hoặc dữ liệu nhập vào trùng từ khóa.

## Giải thích **internal**, **partial**, **public**

| **internal** | **partial** | **public** |
|---|---|---|
| - 1 Class mặc định khi tạp ra là để **INTERNAL**, che giấu ko cho xuất hiện bên ngoài namespace nó đang ở<br>- Ko gõ từ **INTERNAL** đc gọi là **DEFAULT**, giấu luôn<br>- Vậy nếu mún **PUBLIC CLASS** vượt ra khỏi căn phòng, vượt ra khỏi namespace, ta để keyword **PUBLIC**<br>- Che giấu, mở ra, đc gọi chung **ENCAPSULATION**<br>- Và **public**/ **private**/ **protected**/ **default** - ko gõ từ khóa/ **internal** đc gọi chung: **ACCESSOR**, **ACCESS MODIFIER**, **ACCESS SPECIFIER**. | - **partial class/struct** cho phép tách định nghĩa ra nhiều file, tiện cho auto-generated code và teamwork.<br>- Tất cả phần `partial` phải cùng namespace, accessibility và generic parameters.<br>- Compiler gộp lại thành một kiểu trong lúc build. | - Dùng cho lớp/thành phần cần truy cập từ project khác, ví dụ **API model** hoặc **utility library**.<br>- Khi khai báo `public`, nhớ cân nhắc **API surface** và **versioning**.<br>- **public** thường đi kèm các thiết kế bảo vệ nội bộ (ví dụ constructor private, method internal) để tránh lộ quá nhiều.

## Using

- Tương đương `import Java.util` bên Java
- Muốn xài đồ của người khác, ở nơi khác thì phải xin phép, khai báo.
- Có thể dùng `using static`, `using alias = Namespace.Class;` để rút gọn tên gọi.
- Các `using` nằm ở đầu file, compiler chỉ nạp namespace cần thiết khi build.
